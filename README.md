# Nesafood-Mobile-Food-App-
Aplikasi mobile pesan makanan dengan sistem self pick-up. Menggunakan Software Android Studio Versi 2025.1.2

🍽️NESAFOOD adalah aplikasi Android (APK) pemesanan makanan dengan konsep self pickup yang dikembangkan untuk Kantin Universitas Negeri Surabaya (UNESA). Aplikasi ini dibuat untuk mengatasi permasalahan antrean panjang di kantin kampus dengan memungkinkan mahasiswa memesan dan melakukan pembayaran terlebih dahulu, lalu mengambil pesanan tanpa harus mengantre.

✨ Fitur NESAFOOD
👤 Pembeli (Mahasiswa)
1. Registrasi dan login pengguna
2. Melihat daftar menu kantin
3. Menambahkan menu ke keranjang
4. Melakukan pembayaran (simulasi Midtrans Sandbox)
5. Melihat status pesanan (diterima, diproses, siap diambil)
6. Melihat riwayat pesanan
7. Mengelola profil pengguna

🧑‍🍳 Penjual (Kantin)
1. Login penjual
2. Mengelola menu (tambah, edit, hapus)
3. Melihat pesanan masuk
4. Memperbarui status pesanan
5. Melihat riwayat pesanan

🛠️ Informasi Aplikasi
Platform: Android (APK)
Pembayaran: Simulasi Midtrans Sandbox
Sofware : Android Studio
Emulator : Pixel 4A
